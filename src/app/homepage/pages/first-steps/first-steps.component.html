시작하기

이 장에서는 Nest의 기본 원리 를 배우게됩니다. Nest 애플리케이션의 핵심 요소에 익숙해지기 위해 입문 레벨에서 많은 부분을 다루는 기능으로 기본 CRUD 애플리케이션을 빌드 할 것입니다.

언어

우리는 TypeScript와 Node.js를 좋아합니다. 이것이 Nest가 TypeScript와 순수 JavaScript 모두와 호환되는 이유입니다. Nest는 최신 언어 기능을 활용하므로 간단한 JavaScript로 프레임워크를 사용하려면 Babel 컴파일러가 필요합니다.

우리가 제공하는 예제에서는 주로 TypeScript를 사용하지만 코드 조각을 바닐라 자바​스크립트 구문으로 변경 할 수는 있습니다.

사전 요구사항

운영체제에 Node.js (> = 8.9.0) 가 설치되어 있는지 확인하십시오.

설치

Nest CLI를 사용하면 새 프로젝트를 간단하게 구축할 수 있습니다. npm이 설치 되어 있다면 OS의 터미널에서 다음 명령어로 새 Nest 프로젝트를 만들 수 있습니다. 


$ npm i -g @nestjs/cli
$ nest new project

project 디렉터리는 여러 중요한 파일을 src/ 디렉터리에 만듭니다.

src
app.controller.ts
app.module.ts
main.ts

규칙에 따라 새로 생성 된 모듈은 각자의 전용 디렉터리에 있어야 합니다.
main.ts 애플리케이션의 시작점. NestFactory 를 사용하여 Nest 애플리케이션 인스턴스를 만듭니다.
app.module.ts 애플리케이션의 루트 모듈인 AppModule을 정의합니다.
app.controller.ts 하나의 라우트가 있는 기본 컨트롤러 샘플.

main.ts 에는 비동기 함수가 포함되어있어 애플리케이션을 부트스트랩 합니다.

{{ 'main' | extension: bootstrapSwitch.isJsActive }} 
{{ bootstrap }}

Nest 애플리케이션 인스턴스를 만들려면 NestFactory을 사용합니다. create() 메소드는 INestApplication 인터페이스를 구현하는 객체를 반환합니다. 다음 장에서 자세히 설명할 메소드를 제공합니다.

애플리케이션 실행하기

설치 프로세스가 완료되면 다음 명령을 실행하여 HTTP 서버를 시작할 수 있습니다. 


$ npm run start

이 명령은 src/main.ts 파일에 정의된 포트로 HTTP 서버를 시작합니다. 애플리케이션이 실행되면 브라우저를 열고 http://localhost:3000/에 접속하세요. Hello world!를 볼 수 있을 것 입니다.
