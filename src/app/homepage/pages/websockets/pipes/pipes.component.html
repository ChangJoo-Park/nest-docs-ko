Pipes

There's no difference between web sockets pipes and the regular pipes. The only thing you should be aware of is that instead of throwing HttpException, you should use the WsException. Besides, all pipes will be applied only to the data parameter.

Hint The WsException class is exposed from @nestjs/websockets package.
